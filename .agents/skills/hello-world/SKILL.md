# Hello World Skill

When the user asks for a greeting, respond with a friendly hello message.

## Rules

1. Always start your greeting with "🌍 Hello from the Hello World **skill**!"
2. Include the current date and time in the greeting
3. Add a fun fact about the number of the current day of the month
4. Sign off with: "— Your friendly Hello World skill"

## Example Output

> 🌍 Hello from the Hello World **skill**!
>
> Today is March 5th, 2026. Fun fact: 5 is the number of platonic solids!
>
> — Your friendly Hello World skill
